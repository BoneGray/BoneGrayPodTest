# 第一章设计大纲

本文记录当前游戏第一章的内容方向，用于后续寻找美术资源、搭建场景、设计任务和安排开发切片。

这不是最终剧本，而是第一版可玩的章节蓝图。后续如果故事、资源或玩法目标变化，以本文为基础继续修订。

## 核心方向

游戏第一阶段从“普通僵尸生存”开始，但最终指向更大的真相：这个世界可能不是自然形成的末日，而是某种测试场。僵尸、病患和后续更异常的敌人，可能都是不同进化版本、病毒泄露或高维干预的结果。

第一章的目标不是立刻解释全部世界观，而是让玩家先经历：

```text
陌生地点醒来 -> 寻找安全区 -> 进入家园 -> 获得医院线索 -> 前往医院入口
```

## 第一章目标

第一章暂定为：

```text
家园外围 -> 找到安全区 -> 接到医院线索 -> 抵达医院入口
```

玩家开局不是从家里出发，而是莫名其妙出现在家园附近的随机位置。玩家最初不知道自己在哪里，也不知道发生了什么，只能先寻找相对安全的区域。

找到家园后，玩家获得第一个明确目标：附近有一个类似医院的地方，那里可能与大量僵尸、病患或异常感染有关。第一章结尾以玩家抵达医院入口或获得进入医院的条件作为收束。

## 三个主场景定位

### 家园 / 安全区

家园是第一章的目标点，也是早期玩家理解世界和恢复秩序感的地方。

功能定位：

- 新手阶段的安全区。
- 补给、存档、任务信息和后续升级系统的入口。
- 用来承接玩家从“迷路求生”到“主动调查”的转变。

情绪定位：

- 相对安全，但不是完全安全。
- 外部混乱，内部有简易秩序。
- 应该让玩家产生“这里暂时能喘口气”的感觉。

资源需求：

- 居民区、围栏、临时路障、灯光、仓库。
- 简易医疗点、工作台、储物箱、留言板。
- 可交互 NPC 或替代信息源，例如纸条、无线电、公告板。

### 医院 / 异常源头

医院是第一章之后的主要调查方向，也是故事从“普通灾难”进入“异常实验”的过渡场景。

功能定位：

- 第一阶段主要探索副本。
- 提供感染、病患、实验记录、异常样本等线索。
- 引出更深层区域或未知文明线索。

情绪定位：

- 压迫、混乱、失控。
- 医疗设施和实验设施混杂，给人一种“这里不是普通医院”的感觉。

资源需求：

- 病床、手术室、急诊区、实验室、药柜。
- 破损门、血迹、尸袋、隔离帘、电脑、档案柜。
- 钥匙卡、病例文件、实验记录、药品。
- 医院病患型敌人或带有实验特征的敌人。

### 地道 / 外星文明 / 未知区域

未知区域是后续章节的世界观升级点，不建议第一章直接展开完整玩法，但可以在医院线索中提前埋伏笔。

功能定位：

- 揭示世界不是普通末日。
- 承接“病毒泄露 / 进化测试 / 高维观察”的核心反转。
- 后续投放更高级敌人、特殊材料和非现实场景机制。

情绪定位：

- 陌生、非人类、规则感异常。
- 让玩家感觉自己正在接近一个不该存在的地方。

资源需求：

- 地下通道、异星材料、发光结构、培养舱。
- 未知符号、控制台、机械装置、非自然地形。
- 进化版本敌人、实验体、异常植物或生物结构。

## 第一章可玩流程

第一章建议先做成一个小而完整的闭环：

1. 玩家在家园外围随机位置出生。
2. 玩家沿途寻找安全区，途中学习移动、拾取、攻击、换弹。
3. 玩家遇到少量敌人和基础可拾取物。
4. 玩家抵达家园，获得短暂安全感。
5. 家园提供第一个明确目标：去医院寻找药品、资料或幸存者线索。
6. 玩家从家园出发，到达医院入口。
7. 第一章结束，下一阶段进入医院内部探索。

## 第一章资源优先级

后续寻找美术资源时，优先按以下顺序补齐：

1. 家园外围和安全区地形：草地、土路、公路、围栏、建筑外观。
2. 家园道具：箱子、路障、灯、工作台、医疗点、公告板。
3. 医院外部资源：医院建筑、入口、破损门、救护车、标识。
4. 基础拾取物：药品、弹药、钥匙卡、文件、武器。
5. 第一章敌人：普通僵尸、病患型敌人、轻度变异敌人。
6. 线索资源：纸条、电脑、档案柜、实验记录图标。

## 第一章暂不展开

以下内容先不在第一章完整展开，避免范围过大：

- 完整医院内部大型地图。
- 地道或外星文明完整场景。
- 大量 NPC 对话系统。
- 完整背包 UI 和复杂任务系统。
- 多结局或复杂剧情分支。
- 大规模开放世界。

这些内容可以先作为线索、门禁、封锁区域或远景出现。

## 待讨论问题

后续需要逐步确认：

- 玩家为什么会出现在家园外围，是失忆、传送、逃亡，还是被投放到测试场？
- 家园里是否有人，还是只有遗留信息？
- 医院目标是找药、找人、找档案，还是找进入下一场景的钥匙？
- 第一章是否需要一个小 Boss，还是只做普通敌人压力？
- 第一章结束条件是进入医院，还是拿到医院线索后返回家园？
- 未知文明更偏外星科技、高维观察，还是生物进化实验？
